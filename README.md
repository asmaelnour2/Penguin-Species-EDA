# Penguin-Species-EDA
# Palmer Penguins: Exploratory Data Analysis 🐧📊

This project performs a comprehensive Exploratory Data Analysis (EDA) on the **Palmer Penguins** dataset. Using Python's powerful visualization libraries, it explores the biological patterns, physical measurements, and distribution of different penguin species (Adelie, Gentoo, and Chinstrap).

## 🚀 Project Overview
The goal is to analyze physical traits such as bill length, flipper length, and body mass to identify distinct characteristics between species and islands. The project involves data merging, cleaning, and advanced statistical visualization.

## ✨ Key Features
* **Multi-Source Data Integration:** Merging multiple datasets (Adelie, Gentoo, and Chinstrap tables) into a unified dataframe.
* **Statistical Analysis:** Correlation heatmaps to identify relationships between physical attributes.
* **Interactive Visualizations:** Dynamic charts using Plotly Express for deeper data exploration.
* **Professional Plotting:** Customized Seaborn and Matplotlib charts with tailored color palettes (e.g., Soft Green-Blue heatmaps).

## 🛠️ Technical Stack
* **Language:** Python
* **Data Handling:** Pandas, NumPy
* **Visualization:** Seaborn, Matplotlib, Plotly Express
* **Environment:** Jupyter Notebook

## 📈 Analysis Highlights
1.  **Species Comparison:** Visualizing differences in body mass and flipper length across species.
2.  **Correlation Heatmap:** Identifying how bill dimensions relate to overall body size.
3.  **Data Cleaning:** Handling missing values and formatting categorical data for analysis.

## 📂 Project Structure
* `penguins.ipynb`: The primary notebook containing data processing and visualization code.
* `README.md`: Documentation for the project.
